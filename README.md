## Run Locally

Clone the project

```bash
   git@github.com:HlaingMinThan/React-Query-Simple-Project.git react-query
```

Go to the project directory

```bash
  cd react-query
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

and then 

u can check on [(http://localhost:3000)](http://localhost:3000)
